## Concepts to be covered:

1. Chat models
2. Prompt templates
3. Chains
4. RAG
5. Agents and Tools

## How to run:

```bash
# 1. Create an .env
python3 -m venv env
source env/bin/activate

# 2. Set the ENVs in .env
cp .env.sample .env

# 2. Install the packages
pip3 install -r requirementst.xt

# 3. Select the relevant section and run the script
```
